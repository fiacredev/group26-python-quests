# Group 26: Python Quests

A collection of beginner Python quests built to strengthen our programming
foundations. The challenges span 6 levels, from printing text to building a
small text-based adventure game.

## Levels

| Level | Theme | Quests |
|-------|-------------------------------------------------|--------|
| 1 | The Magic of Words and Numbers (Variables & Printing) | 1–5 |
| 2 | A Dialogue with the Machine (User Input & Basic Math) | 6–10 |
| 3 | The Crossroads of Logic (Conditional Statements) | 11–15 |
| 4 | The Power of Repetition (Loops) | 16–20 |
| 5 | The Alchemist's Lab (Functions) | 21–24 |
| 6 | The Grand Challenge | 25–30 |

> Each member attempts at least 4 tasks from each level.

## Project Structure

```
group26-python-quests/
├── quests/
│   ├── quest_01_first_spell.py
│   ├── quest_11_guardian_of_the_bridge.py
│   ├── ... (one file per quest)
│   └── quest_30_reflective_scribe.py
└── README.md
```

## Requirements

- Python 3 installed. Check with:

```bash
python3 --version
```

## How to Run

Run any quest from the project root:

```bash
python3 quests/quest_11_guardian_of_the_bridge.py
```
